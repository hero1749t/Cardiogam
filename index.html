How Our System Works

End-to-end ECG monitoring from hardware to cloud

Step 1: Hardware (ECG Device)

Captures ECG signals using Heart BioAmp Candy + nRF52840 microcontroller with medical-grade precision.

â†“

Step 2: Bluetooth Streaming

Sends ECG data wirelessly via BLE in real-time at 60 FPS for smooth visualization.

â†“

Step 3: Flutter Mobile App

Displays ECG graph, computes BPM using Pan-Tompkins algorithm, and exports PDF reports.

Step 4: Supabase Cloud

Securely stores all ECG sessions with encrypted data and instant sync across devices.

Step 5: Doctor Dashboard

Doctors view reports, monitor patients remotely, and receive AI-based arrhythmia alerts.
